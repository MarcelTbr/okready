Welcome to OKReady

OKReady is a simple app to create and update your OKRs whether you are a single person or a team.

OKR (Objectives and Key Results) is a goal system used by Google and others. It is a simple tool to create alignment and engagement around measurable goals.

Objectives are memorable qualitative descriptions of what you want to achieve. Objectives should be short, inspirational and engaging. An Objective should motivate and challenge the team.

Key Results are a set of metrics that measure your progress towards the Objective. For each Objective, you should have a set of 2 to 5 Key Results. More than that and no one will remember them.

Each Objective has a number of Wins set. It's there so you can know if you did not reach, achieved or surpassed the Objective.

Every Objective's Key Result will help you achieve your objective.
Each time you complete a Key Result, the Objective's Win Counter will increase. You can have different Win Ratios for every Key Result, even negative ones.

Wins Ratio is going to multiply the completion of the Key Result by a number, so that you can have Key Results that count twice, ten times (or as many times as you want) more than the others.

You can set objectives for each quarter or Semester. The year will be set automatically.

This is a demo app, you can enter and create, view or edit semesters for the current year, just typing in "guest" as a password.

It's made as simple as possible, hope you enjoy it.
